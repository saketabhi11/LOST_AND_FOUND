## Installation

### Step 1: Clone the Repository

```bash
git clonehttps://github.com/saketabhi11/LOST_AND_FOUND
cd lost-and-found
```
### Install server dependencies
```console cd server
cd backend
npm install
```
### Install client dependencies
```console
cd frontend
npm install
```
### Start the server
```console
cd backend
npm start
```
### Start the client
```console
cd frontend
npm start
```
